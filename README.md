# File-Organizer

A Python script that organizes files into folders based on their file extensions.

## Example

Before:

<pre>Downloads/
├── photo.jpg
├── notes.txt
├── song.mp3</pre>

After:

<pre>Downloads
├── jpg/
│   └── photo.jpg
├── txt/
│   └── notes.txt
├── mp3/
│   └── song.mp3</pre>

Enter the folder path when prompted.
