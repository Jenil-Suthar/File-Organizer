# File-Organizer

A Python script that organizes files into folders based on their file extensions.

## Example

Before:

Downloads/<br>
├── photo.jpg<br>
├── notes.txt

After:

Downloads/<br>
├── jpg/<br>
│   └── photo.jpg<br>
├── txt/<br>
│   └── notes.txt<br>

Enter the folder path when prompted.
